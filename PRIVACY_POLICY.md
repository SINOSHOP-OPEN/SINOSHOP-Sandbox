# Privacy Policy

## 核心原则

SINOSHOP-Sandbox 尊重每一位参与者的隐私与署名自主权。

## 我们承诺

1. **不预设署名**：除非参与者本人通过 PR 或 Issue 主动要求，我们不会在任何公开文档中列出具体机构或人员名称。

2. **占位符优先**：所有"合作机构""已加入节点"等列表均采用占位符，由参与者自行替换。

3. **署名需主动请求**：如需在 README 中展示机构名称，请在 PR 描述中明确注明。

4. **匿名参与**：参与者可随时选择匿名提交。

## 数据收集

本仓库不主动收集任何参与者信息。GitHub 平台本身会记录 PR、Issue 的元数据，这些由 GitHub 的隐私政策管辖。

## 举报与删除

如发现未经您同意的署名出现在本仓库任何位置，请提交 Issue（标题：`[Privacy] Unauthorized attribution`）。

## 无承诺声明

本仓库所有内容按"原样"提供，不附带任何明示或暗示的担保。

我们不承诺：
- 特定时间内的 Issue/PR 响应
- 特定性能指标
- 与任何参与者的一对一互动

我们提供：
- 可自由下载/运行的代码与数据集
- 分级公开的算法文档与仿真报告
- 一个让参与者相互学习、共同进步的公共平台
